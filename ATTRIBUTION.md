# NeuroWriter

> Offline-first AI writing assistant with global hotkey launcher, clipboard pipeline, and system tray quick actions.

NeuroWriter is a privacy-focused AI writing assistant built on top of [NextChat](https://github.com/ChatGPTNextWeb/NextChat). It adds desktop-native features for seamless AI-powered writing workflows.

## Features

- **Global Hotkey** — `Ctrl+Shift+A` to instantly show/hide the app from anywhere
- **Clipboard Pipeline** — Highlight text, press the hotkey, and let AI process it automatically
- **System Tray Quick Actions** — Right-click the tray icon to Summarize, Rewrite, Translate, Fix Grammar, or Explain clipboard contents
- **Enhanced Prompt Library** — Categories, tags, favorites, and JSON import/export for prompt templates
- **15+ LLM Providers** — OpenAI, Claude, Gemini, Ollama (local), and more
- **Minimize to Tray** — Stays running in the background, always one shortcut away
